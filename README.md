# ParentalGuard

Система родительского контроля для домашнего ПК.

Приложение мониторит активность ребёнка и отправляет информацию родителю в Telegram.

---

## Управление ботом

* 👁️ Активное окно — что сейчас открыто на экране
* 🎮 Запущенные игры — список запущенных игр
* 📊 Live Отчёт — актуальная статистика
* 🌐 Браузер — мониторинг браузеров и вкладок
* 📸 Сделать скрин — мгновенный скриншот экрана
* 📸 Последние скриншоты — просмотр сохранённых скринов

---

## Live Отчёт

* Обновление данных в реальном времени
* Отображение текущего активного окна
* Список запущенных игр
* Количество сделанных скриншотов
* Общая статистика активности

---

## Мониторинг браузеров

* Просмотр запущенных браузеров (Chrome, Edge)
* Последние открытые вкладки
* Пагинация вкладок
* Кнопки для быстрого просмотра ссылок

---

## Система уведомлений

Анализ основан на следующих параметрах:

* Запуск игр
* Смена активного окна
* Подозрительная активность
* Превышение времени в приложениях (в разработке)

---

## Основные возможности

- Реальное время мониторинга
- Автоматические и ручные скриншоты
- Уведомления при запуске игр
- Чтение истории вкладок браузеров
- Удобное управление через Telegram

## ENG

# ParentalGuard

Parental Control System for Windows.

Monitors child's computer activity and sends reports to parent via Telegram.

---

## Bot Controls

* 👁️ Active Window — what is currently open
* 🎮 Running Games — list of launched games
* 📊 Live Report — current statistics
* 🌐 Browser — browser monitoring and tabs
* 📸 Make Screenshot — instant screenshot
* 📸 Last Screenshots — view saved screenshots

---

## Live Report

* Real-time data update
* Current active window display
* List of running games
* Screenshot statistics
* Overall activity overview

---

## Browser Monitoring

* Shows running browsers (Chrome, Edge)
* Last opened tabs
* Pagination support
* Quick link opening buttons

---

## Features

- Real-time activity monitoring
- Game detection and notifications
- Automatic and manual screenshots
- Browser tab monitoring
- Clean and convenient Telegram interface
